返回符合条件的数组

```javascript
const words = ['spray', 'elite', 'exuberant', 'destruction', 'present'];

const result = words.filter((word) => word.length > 6);

console.log(result);
// Array ["exuberant", "destruction", "present"] 
console.log(words);
// Array ["spray", "elite", "exuberant", "destruction", "present"]
 
```
